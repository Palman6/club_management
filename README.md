# README

## Initial Setup   

**Prerequisites**  

The setups steps:  
    - Git  
    - Ruby 3.1.4  
    - Rails 7.1.3  
    - sqlite3   
    - graphql 

**1. Check out the repository and install the needed gem** 

```
git clone git@github.com:Palman6/club_management.git
cd club_management
bundle install
```

**2. Execute test** 
```
 rspec
```

**3. Check Grapgql API** 
```
http://localhost:3000/graphiql
```
